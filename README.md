# Usage

## Step 1

在浏览器中打开storingSelection.html

## Step 2

鼠标选中一些文本，然后把鼠标移动到按钮"mouse here to store current selection"上

## Step 3

刷新浏览器，你将看到选中的文中会再次处于选中状态。你可以不断刷新页面看看
